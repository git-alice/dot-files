# Use Zinit for Plugin Management

The Zsh Dotfiles use Zinit as the plugin manager because the setup should be reproducible, fast, and flexible while staying focused on Zsh only. Simpler options such as Homebrew-sourced plugins or Antidote were considered, but Zinit better fits a personal shell configuration that may later grow lazy loading, plugin reports, direct GitHub plugins, and macOS/VPS bootstrap support.
